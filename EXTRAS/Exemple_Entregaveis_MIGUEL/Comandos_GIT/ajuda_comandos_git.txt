(nao da para copiar e colar os comandos porque têm de ser introduzidos em letra minuscula(pequena) (foi de proposito!))

1. COPIAR ENDEREÇO DO REPOSITORIO EM FORMATO HTTPS CLICANDO NO BOTÃO CLONE

2. IR PARA A PASTA DESEJADA E ABRIR LINHA DE COMANDOS

3. GIT INIT

4. GIT REMOTE ADD ORIGIN ENDEREÇO_COPIADO_DA_ALINEA_1

5. GIT PULL ORIGIN MASTER

6. CRIAR NOVO BRANCH PARA FAZER ALTERAÇÕES COM A SEGUINTE CONVENÇÃO - 
	 GIT CHECKOUT -B branch_NomeMembro_Porque
EXEMPLO: git checkout -b branch_Miguel_Documentacao

7. FAZER ALTERAÇÕES ...

8. MUDAR PARA BRANCH MASTER - GIT CHECKOUT MASTER

9. FAZER MERGE - GIT MERGE NOME_BRANCH_CRIADO_NA_ALINEA_6

10. OPCIONAL PODEM APAGAR OU NÃO O BRANCH CRIADO, EU APAGO SEMPRE DEPOIS DO MERGE, E ANTES DE COMEÇAR ALTERAÇÕES CRIO NOVO BRANCH NOVAMENTE: 
	COMANDO PARA APAGAR:- GIT BRANCH -D NOME_BRANCH_CRIADO_NA_ALINEA_6

11. SUBMETER ALTERAÇÕES PARA REPOSITORIO REMOTO:- GIT ADD . (O ".PONTO" ADICIONA TUDO, SE QUISEREM APENAS ALGUM FICHEIRO COLOQUEM O NOME DO FICHEIRO EM VEZ DO PONTO)

12. GIT COMMIT -M "MENSAGEM_A_SER_ENVIADA_NESTE_COMMIT"

13. GIT PUSH ORIGIN MASTER

DONE!! 



OPCIONAL:

	- IR DOCUMENTANDO O QUE EU FIZ NUM FICHEIRO .TXT PARA SER FACIL DEPOIS FAZER RELATORIO INDIVIDUAL
